SETUP

1. install local postgresql on your developing machine
    https://www.postgresql.org/download/
2. create database with following details:
    db name: lab1
    user: postgres
    password: 1234
    port: 5432
3. install node_modules of server
    'cd server'
    'npm i'
4. start server
    'npm run start:dev'
5. Launch Frontend by just inspecting login.html
6. That's it!
